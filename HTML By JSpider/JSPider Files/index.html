Web Technology Intoduction

  • Software Definition: It is a collection of multiple programs which developed to solve the real time problem.

  • Browser Definition: It is a software which is used to fetch the information from WWW (World Wide Web) according to the user need.

  • Internet: It is a wide network which helps to connect multiple devices through out the world.

  • Website: It is a collection of multiple webpages.

    Ex: Facebook, Amazon, Google etc.

  • Web Page: Each and every individual pages in a website is called web pages.

    1. Static Web Page: Any web page which will display same content for different different users.

      Ex: Contact Us, Terms & Condition, About etc

    2. Dynamic Web Page: Any web page which will display different content for different different users.

      Ex: Facebook Home Page, WhatsApp Home Page etc.

  • Server: It is a combination of Hardware and Software which is used to store the data.

  • Web Technology: It refers to the verious tools and technique which is used to develop web pages.

HTML:

  • HTML stand for Hyper Text Markup Language.
  • It is used to develop Web Pages.
  • It is a simple languge where we have to use the tags and execute the code in one of the browser.

    The language attribute (html lang="en") decleares the primary language of the page using the ISO language code.
