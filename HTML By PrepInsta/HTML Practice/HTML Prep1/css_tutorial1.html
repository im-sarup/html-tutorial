CSS or Casecading Style Sheet

CSS: It is a stylesheet language used to describe the look and formatting of a document written in HTML.

    CSS allows you to control the appearance of webpages by specifying styles for elements, such as colors, fonts, layouts, and spacing

    CSS rules consist of two main components: Selectors and Declarations

      i. Selectors: Selectors are used to target specific HTML elements
      ii. Deselectors: While declarations define the styles to be applied to those elements.
  • Notes: A declaration consists of a property (the aspect of the element being styled) and a value (the specific style to apply). CSS rules are enclosed within curly braces ({}) and are usually placed in a separate CSS file or within the <"style"> element in an HTML document.

    CSS Style Methods

    CSS can be applied to HTML documents in three different ways. Each method has its own use cases and advantages. They are:
      1. Inline Style
      2. Internal Style
      3. External Style

    1. Inline Style

    CSS styles are added directly to the HTML element using the style attribute.

    This method is not recommended, as it can lead to messy code and makes it difficult to maintain and update styles.

    2. Internal Style

    CSS rules are placed within a <"style"> element in the <"head"> section of the HTML document. This method is suitable for small projects or single-page websites.

    3. External Style

    CSS rules are placed in a separate CSS file (e.g., styles.css) and linked to the HTML document using the <"link"> element.

    This method is recommended for larger projects or multi-page websites, as it keeps the styles separate from the HTML, making it easier to maintain and update.
